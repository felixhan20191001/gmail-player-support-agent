The latest version of the game now supports cloud save. To move your progress to a new phone, please update the game to the latest version first, then bind your Facebook account on your current device.

After that, install or open the game on the new device and log in with the same Facebook account. Your saved progress should sync to the new device.

Please make sure the progress has synced successfully before uninstalling the game or clearing data from the old device.
